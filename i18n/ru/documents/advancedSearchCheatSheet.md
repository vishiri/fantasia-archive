- Примечания к типам фильтров
  - Замените пробелы на `-`
  - Префикс `@` для включения в поиск `Другие имена`
- Типы фильтров
  - `$` - Тип документа
  - `#` - Тег
  - `>` - Иерархический путь
  - `^` - Переключить
    - `^c` — это категория
    - `^d` - мертв/ушел/уничтожен
    - `^f` - Закончено
    - `^m` — второстепенный документ
- Полный поиск
  - `%` - Начало полного поиска
  - `:` - Разделитель между именем поля и значением поля.
  - `%some-field:some-value` — поиск значения по всем полям.
  - Это также можно использовать для поиска цветов: `%color:blue`
  - `""` перенос для точного поиска (имя поля и/или значение)
    - Полный пример: `%"some-field":"some-value"`
    - Пример точного значения: `%some-field:"some-value"`
    - Точное имя примера: `%"some-field":some-value`
