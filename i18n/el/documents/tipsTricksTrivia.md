# Συμβουλές, κόλπα & Trivia
----------

- Πληκτρολογώντας «@» ενώ επεξεργάζεστε μεγάλα πεδία κειμένου, μπορείτε να συνδεθείτε με άλλα έγγραφα απευθείας μέσα στο πρόγραμμα επεξεργασίας κειμένου!
- Το `Ctrl+Shift+F` ανοίγει την εύρεση στο τρέχον έγγραφο όταν χρειάζεστε μια συγκεκριμένη λέξη στη σελίδα!
- Οποιοδήποτε πεδίο στη λειτουργία επεξεργασίας εγγράφων με το μικρό αναπτυσσόμενο βέλος επιτρέπει προσαρμοσμένες τιμές! Κλείστε το μενού των προκαθορισμένων τιμών με Esc, πληκτρολογήστε την επιθυμητή τιμή και επιβεβαιώστε με Enter.
- Μπορείτε να κάνετε αναζήτηση σε όλα τα έγγραφα και όλα τα πεδία τους ταυτόχρονα χρησιμοποιώντας τα ακόλουθα είτε στο αναδυόμενο παράθυρο Γρήγορη αναζήτηση είτε σε οποιαδήποτε από τις αναζητήσεις σχέσεων: `%field-name:whatever-you-need`
- Υπάρχει μια ολόκληρη λίστα με πραγματικά χρήσιμα keybinds που μπορούν να κάνουν τη χρήση του FA πολύ πιο γρήγορη και ευκολότερη!
- Το μενού ρυθμίσεων περιέχει μια ολόκληρη ποικιλία από μεγάλες και μικρές τροποποιήσεις για να προσαρμόσετε την εφαρμογή στις ανάγκες σας!
- Το FA έχει μια Dark Mode που δεν είναι απλή εκ των υστέρων σκέψη και είναι πλήρως επισκευή!
- Πολλά μέρη της εφαρμογής αντιδρούν σε ένα μεσαίο κλικ του ποντικιού. Για παράδειγμα, μπορείτε να κλείσετε καρτέλες εγγράφων ή να ανοίξετε ιεραρχικά στοιχεία δέντρου απευθείας με αυτό!
- Το Fantasia Archive είναι δωρεάν. δεν υπάρχουν συνδεδεμένες συμβολοσειρές: Δεν υπάρχουν διαφημίσεις, δεν υπάρχουν ιχνηλάτες, δεν γίνεται εξόρυξη κρυπτονομισμάτων στον υπολογιστή σας και τα δεδομένα σας παραμένουν αποκλειστικά δικά σας!- Εάν αυτές οι συμβουλές σας ενοχλούν, υπάρχει ένας διακόπτης για να τις απενεργοποιήσετε στις ρυθμίσεις της εφαρμογής.
- Η αναζήτηση σχέσεων είναι γεμάτη με επιπλέον δυνατότητες που μπορείτε να χρησιμοποιήσετε για να βρείτε αυτό που χρειάζεστε! Διαβάστε τα πάντα στον «Οδηγό σύνθετης αναζήτησης» στο μενού Βοήθεια!
- Το έργο σας αποθηκεύεται αυτόματα κατά την έξοδο, αλλά συνιστάται να χρησιμοποιείτε τη λειτουργία «Αποθήκευση» κάθε τόσο για να βεβαιωθείτε ότι τα δεδομένα παραμένουν ασφαλή.
- Το αριστερό ιεραρχικό δέντρο μπορεί να επεκταθεί σύροντας το διαχωριστικό και η εφαρμογή θυμάται πού το αφήσατε την επόμενη φορά που θα ξεκινήσετε!
- Μη διστάσετε να προσθέσετε όσες υποκατηγορίες θέλετε. Μπορείτε πάντα να μετακινήσετε πράγματα αργότερα και να αναδιοργανώσετε τη δομή του έργου σας.
- Οι ετικέτες είναι μια πολύ ισχυρή δυνατότητα που σας επιτρέπει να προσθέσετε ημικατηγορίες στο δέντρο του έργου και να δημιουργήσετε πρόσθετες παραμέτρους αναζήτησης για να βοηθήσετε την πλοήγηση.
- Υπάρχει ένας οδικός χάρτης προγραμματισμένων λειτουργιών στο Patreon! Αν είστε περίεργοι για το τι θα έρθει τελικά, μη διστάσετε να το διαβάσετε.
- Αυτό το λογισμικό είναι στην πραγματικότητα ένας τοπικός ιστότοπος που προσποιείται ότι είναι μια εφαρμογή επιτραπέζιου υπολογιστή! Η τεχνολογία είναι τρελή!
- Αν χρειαστείτε ποτέ ένα καλό λογισμικό δημιουργίας χαρτών εκτός σύνδεσης, δείτε τα Wonderdraft και Dungeondraft της Megasploot!- Ο αξιολάτρευτος μικρός απόκρυφος δράκος που μπορεί να έχετε δει κατά τη χρήση της εφαρμογής ονομάζεται Fantasia και είναι η επίσημη μασκότ του FA!
- Αν ποτέ θελήσετε να χορηγήσετε την ανάπτυξη, υπάρχει Patreon. Εάν δεν μπορείτε ή δεν θέλετε, αυτό είναι επίσης εντάξει—απλώς διασκεδάστε με την εφαρμογή και φτιάξτε κάτι υπέροχο με αυτήν!
- ΑΠΟΠΟΙΗΣΗ ΕΥΘΥΝΗΣ: Δεν τραυματίστηκαν μαγικά πλάσματα κατά την κατασκευή αυτού του λογισμικού.
