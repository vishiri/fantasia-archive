# ヒント、コツ、トリビア
----------

- 大きなテキスト フィールドの編集中に「@」を入力すると、テキスト エディター内で他のドキュメントに直接リンクできます。
- ページ上で特定の単語が必要な場合、「Ctrl+Shift+F」は現在のドキュメント内で検索を開きます。
- ドキュメント編集モードの小さなドロップダウン矢印のあるフィールドでは、カスタム値を使用できます。 Esc キーで定義済みの値のメニューを閉じ、希望の値を入力して Enter キーを押して確定します。
- クイック検索ポップアップまたは任意の関係検索で次のコマンドを使用して、すべてのドキュメントとそのフィールドを一度に検索できます: `%field-name:whatever-you-need`
- FA をより迅速かつ簡単に使用できる、非常に役立つキーバインドのリストがすべてあります。
- 設定メニューには、アプリをニーズに合わせて調整するための大小さまざまな調整が含まれています。
- FA にはダーク モードがあり、単なる思いつきではなく、十分に活用できます。
- アプリの多くの部分が中マウスのクリックに反応します。たとえば、ドキュメントのタブを閉じたり、階層ツリー項目を直接開いたりすることができます。
- Fantasia Archive は無料です。なんの制約もありません: 広告もトラッカーも、PC 上で暗号通貨マイニングも発生せず、データは完全にあなたのものになります。- これらのヒントが気になる場合は、アプリの設定にヒントをオフにするスイッチがあります。
- 関係検索には、必要なものを見つけるために使用できる追加機能が満載です。詳細については、[ヘルプ] メニューの「高度な検索ガイド」を参照してください。
- プロジェクトは終了時に自動保存されますが、データが安全に保たれていることを確認するために、時々「保存」機能を使用することをお勧めします。
- 左側の階層ツリーは分割線をドラッグすることで展開でき、アプリは次回起動時にどこから離れたかを記憶します。
- サブカテゴリを必要なだけ追加することを躊躇しないでください。後でいつでも内容を移動したり、プロジェクトの構造を再編成したりできます。
- タグは、プロジェクト ツリーにセミカテゴリを追加し、ナビゲーションを支援する追加の検索パラメータを作成できる非常に強力な機能です。
- Patreon には予定されている機能のロードマップがあります。最終的にどうなるのか気になる方はぜひ読んでみてください。
- このソフトウェアは実際にはデスクトップ アプリケーションを装ったローカル Web サイトです。テクノロジーはクレイジーです！
- 優れたオフライン地図作成ソフトウェアが必要な場合は、Megasploot の Wonderdraft と Dungeondraft を見てください。- アプリの使用中に見たことのある愛らしい小さな難解なドラゴンはファンタジアと呼ばれ、FA の公式マスコットです。
- 開発のスポンサーになりたいと思ったら、Patreon があります。それができない、またはしたくない場合でも、それはまったく問題ありません。アプリを楽しんで、それで何かクールなものを構築してください。
- 免責事項: このソフトウェアの作成中に魔法の生き物は傷つけられませんでした。
