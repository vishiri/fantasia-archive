# Traduction Française pour Fantasia Archive

## 📝 Description

Cette contribution ajoute une traduction française complète à Fantasia Archive, permettant aux utilisateurs francophones de profiter pleinement de cette application de worldbuilding.

## 🌍 Comment utiliser cette traduction

Une fois intégrée à l'application, les utilisateurs pourront :
1. Aller dans les paramètres de Fantasia Archive
2. Sélectionner "Français" comme langue
3. L'interface sera entièrement en français

## 🔍 Notes de traduction

### Choix de traduction
- **"World-building"** → "création d'univers" ou conservé tel quel selon le contexte
- **"Document"** → "document" (conservé tel quel)
- **"Project"** → "projet"
- **"Switch"** → "interrupteur"
- **"Tag"** → "étiquette"
- **"Minor document"** → "document mineur"

### Particularités techniques
- Les symboles de recherche (`@`, `$`, `#`, `>`, `^`, `%`, `:`) sont conservés tels quels car ils font partie de la syntaxe de recherche
- Les termes techniques comme "hierarchical path", "full-search", etc. sont traduits de manière cohérente dans toute la documentation
